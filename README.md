<h1 align="center">Проект "Mesto"</h1>

Проект на основе HTML5, CSS3, Javascript.  
Файловая структура по БЭМ.

`HTML`+`CSS`:
* Флекс-бокс
* Грид-сетка
* Медиа-запросы
* Адаптивная вёрстка


`JavaScript` :
* Открытие попап при нажатии кнопки "Редкатировать".
* Закрытия попап при нажатии кнопки кнопки "X"
